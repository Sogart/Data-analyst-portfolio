# Online Retail Sales Analysis

## Overview
This project analyzes transactional retail data to understand sales performance,
customer behavior, and product contribution to revenue.

## Objectives
- Calculate total revenue and order volume
- Identify top products and customers
- Analyze revenue trends over time

## Tools Used
- Python (pandas, matplotlib)

## Key Insights
- Revenue is driven by a small group of high-value products
- Customer spending is highly uneven
- Sales show clear seasonal patterns

## Dataset
Online Retail II dataset (public retail transaction data)
