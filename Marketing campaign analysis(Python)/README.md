# Marketing Campaign Performance Analysis

## Overview
This project analyzes customer response to multiple marketing campaigns to
identify which customer segments respond best to promotions.

## Objectives
- Measure overall campaign acceptance rate
- Analyze response patterns by demographics
- Understand income impact on campaign effectiveness

## Tools Used
- Python (pandas, matplotlib)

## Key Insights
- Campaign response varies significantly across education levels
- Higher-income customers show stronger engagement
- Targeted campaigns can improve conversion rates

## Dataset
Marketing Campaign dataset (public marketing data)
